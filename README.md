Task Management App

This app allows users to add tasks with a name, category, deadline, and status. The tasks are displayed on the page and can be managed dynamically.

Users can add tasks and view them in a list format. The app updates the display each time a task is added or removed. I also implemented local storage so tasks remain saved even after refreshing the page.

This project uses HTML, CSS, and JavaScript, including arrays, objects, and DOM manipulation.
